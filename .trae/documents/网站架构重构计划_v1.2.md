## 五、技术细节

### 5.1 下拉菜单实现

```typescript
// Navbar.tsx 核心逻辑
const [dropdownOpen, setDropdownOpen] = useState(false);

// 桌面端：悬停展开
<div 
  onMouseEnter={() => setDropdownOpen(true)}
  onMouseLeave={() => setDropdownOpen(false)}
>
  <button>
    {t('home')} <ChevronDown className={dropdownOpen ? 'rotate-180' : ''} />
  </button>
  {dropdownOpen && (
    <ul>
      {homeChildren.map(child => <li><a href={child.href}>{t(child.key)}</a></li>)}
    </ul>
  )}
</div>
```

## 六、风险评估

### 6.1 潜在问题
1. **SEO影响**：独立页面可能降低首页权重
   - 解决方案：在首页添加内链指向独立页面
   
2. **用户体验**：页面跳转增加用户操作成本
   - 解决方案：独立页面添加面包屑导航和返回按钮

3. **国际化路由**：next-intl需要正确处理子页面路由
   - 解决方案：使用 `useRouter` 和 `usePathname` 确保locale正确

### 6.2 兼容性
- 移动端下拉菜单需要点击展开（非悬停）
- 保持暗色模式兼容
- 保持响应式布局
