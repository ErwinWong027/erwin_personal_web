# 履历分页样式实现计划

## 📊 需求分析

根据参考图，需要将教育经历和工作经历改造成以下样式：

### 参考图特点
1. **时间轴设计**：左右交替布局，中间有时间轴线连接
2. **彩色卡片**：每个经历使用不同的鲜艳背景色（粉色、蓝色、黄色等）
3. **时间标签**：卡片顶部显示时间段（如 2022.06 - 至今）
4. **内容结构**：
   - 公司名称（加粗）
   - 职位/角色
   - 要点列表（bullet points）
5. **装饰元素**：时间轴上的菱形节点和底部的小图标

## 🎯 实现目标

### 教育经历（3 条）
- 采用时间轴布局
- 每条教育经历使用不同颜色背景
- 显示学校名称、学位、专业、时间段
- 添加关键成就描述

### 工作经历（4 条）
- 采用时间轴布局
- 每条工作经历使用不同颜色背景
- 显示公司名称、职位、时间段
- 以要点列表形式展示工作内容

##  实现步骤

### 步骤 1：创建新的卡片组件
**文件**: `src/components/sections/TimelineCard.tsx`

**功能**:
- 支持可配置的背景颜色
- 时间轴节点样式（菱形/圆形）
- 响应式布局（移动端单列，桌面端左右交替）
- 支持要点列表渲染

**Props**:
```typescript
interface TimelineCardProps {
  title: string;        // 公司/学校名称
  role: string;         // 职位/学位
  period: string;       // 时间段
  description?: string; // 描述文本
  highlights?: string[];// 要点列表
  color: string;        // 背景颜色类名
  position: 'left' | 'right'; // 位置（用于左右交替）
}
```

### 步骤 2：重构 Experience 组件
**文件**: `src/components/sections/Experience.tsx`

**改动**:
1. 引入 `TimelineCard` 组件
2. 将教育经历和工作经历数据改为数组
3. 实现时间轴布局容器
4. 为每个卡片分配不同的背景色
5. 添加时间轴线和装饰元素

**颜色方案**:
- 教育经历：粉色系、紫色系、蓝色系
- 工作经历：黄色系、绿色系、橙色系、青色系

### 步骤 3：更新数据结构
**文件**: `src/data/portfolio-knowledge.ts`

**改动**:
1. 为教育经历添加 `highlights` 字段
2. 为工作经历添加 `highlights` 字段
3. 或者在组件本地定义增强数据

### 步骤 4：添加动画效果
**功能**:
- 卡片滚动进入动画
- 时间轴线绘制动画
- 悬停效果增强

### 步骤 5：响应式优化
**移动端**:
- 单列垂直布局
- 时间轴线居中
- 所有卡片在时间轴右侧

**桌面端**:
- 左右交替布局
- 时间轴线居中
- 卡片宽度自适应

##  样式设计

### 颜色调色板
```css
/* 教育经历 */
edu-1: bg-gradient-135 from-pink-200 to-pink-300
edu-2: bg-gradient-135 from-purple-200 to-purple-300
edu-3: bg-gradient-135 from-blue-200 to-blue-300

/* 工作经历 */
work-1: bg-gradient-135 from-yellow-200 to-yellow-300
work-2: bg-gradient-135 from-green-200 to-green-300
work-3: bg-gradient-135 from-orange-200 to-orange-300
work-4: bg-gradient-135 from-cyan-200 to-cyan-300
```

### 时间轴线
```css
/* 垂直中线 */
timeline-line: absolute left-1/2 transform -translate-x-1/2 w-1 bg-slate-300

/* 节点装饰 */
timeline-node: absolute left-1/2 transform -translate-x-1/2 w-4 h-4 rotate-45 bg-slate-400
```

### 卡片样式
```css
card: relative p-6 rounded-xl shadow-lg border-2 border-slate-800
card-content: text-sm leading-relaxed
card-title: font-bold text-base mb-1
card-role: text-sm opacity-80 mb-2
card-period: text-xs font-semibold opacity-70 mb-3
card-list: list-disc list-inside space-y-1
```

## ⚠️ 注意事项

1. **颜色对比度**: 确保文字在彩色背景上清晰可读
2. **响应式断点**: 移动端 (<768px) 使用单列，桌面端使用双列
3. **内容长度**: 要点列表控制在 3-5 条，避免卡片过长
4. **动画性能**: 使用 CSS transform 而非 position 动画
5. **暗色模式**: 考虑暗色模式下的颜色适配
